# SERVICE PAGE BRIEFS — Vision Motors

**Updated:** 2026-07-22 · **For:** Phase 3

**Production rule (approved):** no page is built from a template swap with only
the service name changed. Each must be independently valuable. Each needs the
nine fields below completed **and** a real photograph before it ships.

**Template:** `components/templates/service-page-template.tsx` — a structure, not
a content generator.

---

## 🚫 Prohibited claims — every page

These are excluded sitewide and repeated here because service pages are exactly
where they would creep back in.

| Never publish | Why |
|---|---|
| Any warranty duration or mileage | C1 unresolved. Only `siteConfig.warrantyInterimCopy` may be used |
| "RMI Approved" · "MIWA Accredited" · "ARASA Member" · "5 Star RMI" · "accredited workshop" | C2 — logo files and a self-claim are not membership evidence |
| **"The 3.2 Ranger has a wet timing belt"** | **False — the P5AT is chain-driven.** This is the error the top-ranking incumbent page makes |
| **"The 3.2 has a balance-shaft problem"** | No credible source. An inline-five is inherently balanced |
| Any Ranger overheating / fire recall | **That is the Ford Kuga story.** Factually wrong and legally risky |
| "11,000-unit 2014 turbo recall" | Uncorroborated at any primary source |
| Any current-model **Mazda BT-50** capability | Post-2020 BT-50 is an Isuzu D-Max twin; discontinued in SA March 2024. Scope to **2011–2020 only** |
| Turnaround times | C11 — a 1-star review cites a 6-week wait |
| Pricing, "from R…", "affordable", "cheapest" | Nothing confirmed; superlatives are a CPA/ASA exposure |
| AdBlue / SCR / DPF removal | Excluded on ethical grounds (E5) |
| Manufacturer approval or authorised-dealer wording | Displaying a brand logo on signage ≠ authorisation |

**Every technical claim on a service page must trace to a row in
`CONTENT-SOURCE-REGISTER.md`.** If it does not, it does not ship.
